Frequently Asked Questions

How do I import my financial data into ORBIS?
You can import financial data using our CSV import feature. Go to Settings > Data Import, select your CSV file, and map the columns to our data fields. We support common formats from Excel, QuickBooks, and other accounting software.
Can I export my reports and analytics?
Yes! All reports and analytics can be exported in multiple formats including PDF, Excel, CSV, and PowerPoint. Use the export button in the top-right corner of any report or chart.
How often is my data updated?
Data is updated in real-time for manual entries and every 15 minutes for automated integrations. You can also manually refresh data anytime using the refresh button in your dashboard.
What security measures are in place to protect my data?
We use bank-level encryption (AES-256), two-factor authentication, regular security audits, and secure cloud hosting. Your data is backed up daily and stored in multiple secure locations.
Can I customize the dashboard layout?
Absolutely! You can drag and drop widgets, resize charts, add or remove data panels, and save multiple dashboard views. Access customization options from the dashboard settings menu.
